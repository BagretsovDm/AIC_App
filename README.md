# Проектная работа курса Android Developer.Basic
## Платформа OTUS

Простое мобильное приложение на ОС Android для просмотра экспонатов музея The Art Institute of Chicago.

## Технологии
- MVVM
- Splash screen
- Navigation component
- OkHttp
- Retrofit
- Moshi
- Picasso
- Room
- Hilt
